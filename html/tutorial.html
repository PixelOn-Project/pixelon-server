PixelOn이란?

PixelOn은 PixelArt Editor인 Piskel에 On-Device 이미지 생성 기능을 추가한 확장 프로그램입니다. 클라우드 구독료 없이, 여러분의 컴퓨터 성능을 활용하여 언제 어디서든 픽셀 아트를 생성할 수 있습니다.

Stable Diffusion C++ 기술을 Piskel에 결합하여 가볍고 빠른 성능을 자랑합니다. 아이들을 위한 컬러링 북 제작부터 전문 픽셀 아트 작업까지, PixelOn으로 표현력을 높여보세요!

시스템 요구사항

원활한 이미지 생성을 위해 아래 사양을 권장합니다.

CPU Intel i5 이상
RAM 16GB 이상
GPU NVIDIA RTX 2000번대 이상 (VRAM 4GB+)
Disk 20GB 이상의 여유 공간
참고: CPU 모드로도 작동은 가능하지만, 이미지 한 장당 2~3분이 소요됩니다. (GPU 사용 시 약 10초) AMD GPU 지원은 준비 중입니다.

설치 및 실행 방법

  1. 제공된 PixelOn Installer를 다운로드하세요.
  2. 설치 프로그램을 실행하면 시스템 하드웨어를 스캔하여 최적의 버전을 자동으로 설치합니다.
  3. 설치가 완료되면 바탕화면의 PixelOn 아이콘을 실행합니다.

사용 가이드 (How to Use)

1. Simple Prompt (간편 생성)

에디터 하단의 프롬프트 창을 이용해 빠르게 이미지를 생성할 수 있습니다. 이곳에는 그리고 싶은 내용을 담은 Positive Prompt만 입력 가능합니다.

2. Detail Dialog (상세 설정)

더 세밀한 조작이 필요하다면 상세 창을 이용하세요.

  • ⓐ History: 이전 생성 기록(세션)을 확인합니다.
  • ⓑ Prompt & Settings: 프롬프트 및 세부 옵션을 설정합니다.
  • ⓒ Results: 생성된 이미지가 나타납니다.
  • ⓓ Log/Export: 진행 로그를 확인하고 이미지를 에디터로 내보냅니다.

3. History (기록 관리)

세션 목록에서 과거 작업물을 불러오거나, ... 버튼을 눌러 세션 이름을 변경/삭제할 수 있습니다.
※ 삭제 시 복구가 불가능하니 주의하세요.

4. Prompt & Settings (프롬프트 설정)

  • Positive Prompt: 생성하고 싶은 장면에 대한 설명 (Enter로 태그 구분)
  • Negative Prompt: 제외하고 싶은 요소에 대한 설명 (Enter로 태그 구분)
  • Preset: Normal(기본), A, B 등 스타일에 맞는 프리셋 선택
  • Seed: 특정 시드값을 고정하여 비슷한 구도의 이미지를 생성 가능

자주 묻는 질문 (Q&A)

Q. NVIDIA GPU인데 CUDA가 선택되지 않아요.
A. GPU 드라이버 버전이 낮아서 발생할 수 있습니다. NVIDIA 공식 홈페이지에서 최신 드라이버로 업데이트해주세요.

Q. 서버 연결 실패 오류가 뜹니다.
A. 설치 중 네트워크 문제로 파일이 누락되었을 수 있습니다. 재설치를 진행해주세요.

Q. 생성이 너무 느립니다.
A. GPU 가속기가 없거나 사양이 낮을 경우 느릴 수 있습니다. 이는 로컬 AI 모델 구동의 특성입니다.
